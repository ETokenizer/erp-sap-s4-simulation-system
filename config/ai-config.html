📊 模型轮转策略

🔄 自动切换已启用

调用失败时自动切换下一个模型。免费额度先到期优先使用，全部耗尽在后使用 Coding Plan 兜底。

模型 到期时间 额度 状态

1. 标准 DashScope（免费额度）

  1. 访问 阿里云 DashScope 控制台
  2. 注册/登录阿里云账号，开通 DashScope 服务
  3. 在 API Key 管理页面创建或获取 API Key
  4. 在上方 Tab 中填写 Base URL 和 Token，选择模型，保存配置

2. Coding Plan（编码计费）

  1. 访问 百炼控制台
  2. 开通 Coding Plan 服务，获取独立 API Key
  3. 在 Coding Plan Tab 中选择协议（OpenAI/Anthropic），填写 Token
  4. Coding Plan 模型无额度限制，按实际使用的 token 量计费

3. CORS 跨域说明

标准 DashScope API（dashscope.aliyuncs.com）支持浏览器 CORS，可直接调用。 Coding Plan API（coding.dashscope.aliyuncs.com）不支持浏览器 CORS，需通过本地服务器或 CORS 代理访问。 如遇到 CORS 错误，请确保使用 http://localhost:8080 启动本地服务器访问。

4. 使用 AI 生成文档

配置完成后，返回主页面，在完成任务时选择"AI 生成"选项，即可调用 AI 生成定制化文档内容。 模型轮转会自动优先使用快到期的免费额度，全部耗尽在后使用 Coding Plan 兜底。
