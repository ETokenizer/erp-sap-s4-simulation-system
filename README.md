# ERP SAP S4 HANA 上线模拟演练系统
> 企业级项目仓库化管理平台 - 基于 SAP ACTIVATE 方法论的策略模拟游戏 (SLG) 系统

## 🚀 快速开始

### 第一步：登录系统
1. 打开 `login.html` 页面
2. 输入任意用户名和密码（模拟登录，无需真实验证）
3. 点击登录进入系统

### 第二步：创建/选择项目
1. 首次使用点击「新建项目」创建新项目
2. 填写项目信息（编号、名称、客户、日期等）
3. 点击项目卡片进入项目空间

### 第三步：开始模拟演练
1. 从第 1 章节开始，依次完成任务
2. 选择手动完成或 AI 生成文档
3. 文档自动归档到对应项目目录

## ⚙️ AI 配置（可选）

### 配置步骤
1. 点击顶部「AI 配置」按钮
2. 填写阿里云 DashScope 配置：
   - **Base URL**: `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
   - **Token**: 在阿里云控制台获取
   - **模型**: qwen-turbo / qwen-plus / qwen-max
3. 点击「测试连接」验证
4. 保存后即可使用 AI 生成文档

### 获取 API Token
访问：https://dashscope.console.aliyun.com/
