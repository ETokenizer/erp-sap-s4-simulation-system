### 特性 6: 中英双语切换

**新建 `scripts/locale.js`**：
- `Locale.t(key)` — 返回当前语言文本
- `Locale.set(lang)` — 设置语言（zh-CN / en-US）
- 约 150 个 key 覆盖全部 UI 文本

**改动**：
- 所有 HTML 硬编码文本改用 `data-i18n` 属性
- 所有 JS 动态文本改用 `Locale.t()`
- Header 增加语言切换按钮（主题切换旁边）
- NPC 气泡 / 角色提示双语

### 特性 7: AI 生成商业化（支付拦截完善）

**当前状态**: ⚠️ **部分实现** - 仅有 `isPaidUser()` 检查函数，缺少完整支付流程。

**新建 `scripts/payment.js`**：
- 产品：ERP AI 文档生成套件 ￥99/项目（37 种文档不限量）
- 支付方式：微信/支付宝（模拟）
- 计费：按项目一次性付费，克隆不继承支付记录

**流程**：
```
点击 AI 生成 → 检查支付 → 未支付 → 弹出支付模态框 → 模拟支付 → 成功 → 继续
```

**预留接口**：`createOrder()`, `payOrder()`, `verifyPayment()`, `getQuota()`, `deductQuota()`
