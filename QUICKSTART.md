# ERP SAP S4 HANA 上线模拟演练系统 - 快速使用指南

## 📌 第一步：登录系统

1. 打开浏览器
2. 访问 `login.html` 文件
3. 输入任意用户名和密码（例如：admin / admin）
4. 点击"登录"按钮

> 💡 说明：系统为模拟登录，无需真实账号验证

---

## 📌 第二步：创建项目

登录后进入项目列表页面：

1. 点击右上角「➕ 新建项目」按钮
2. 填写项目信息：
   - **项目编号**：如 `ERP-2026-001`
   - **项目名称**：如 `XX 公司 SAP 实施项目`
   - **客户/公司名称**：如 `某某集团有限公司`
   - **项目经理**：如 `张三`
   - **项目开始日期**：选择日期
   - **计划上线日期**：选择日期（通常 1 年后）

3. 点击「创建项目」按钮

---

## 📌 第三步：进入项目空间

在项目列表中，点击任一项目卡片即可进入项目空间。

### 项目空间界面布局

```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：项目名称 | 进度条 | 主题切换 | 项目列表                    │
├──────────┬───────────────────────────┬──────────────────────────┤
│  左侧    │  中间                      │  右侧                     │
│  章节列表 │  任务清单                  │  交付物文档               │
│  文件夹树 │  (点击卡片进行操作)        │  当前文件夹文件           │
└──────────┴───────────────────────────┴──────────────────────────┘
```

---

## 📌 第四步：完成任务

1. 从 **第 1 章 战略发现与立项** 开始
2. 点击任务卡片（如"业务痛点分析"）
3. 选择操作方式：
   - **手动完成**：快速标记完成，生成标准模板
   - **AI 生成**：调用 AI 生成定制化内容
4. 点击「完成任务」按钮

### 完成后效果
- ✅ 任务状态变为"已完成"
- 📄 右侧文档列表显示"已生成"
- 📁 文档自动保存到对应项目目录

---

## 📌 第五步：查看文档

### 方式一：从右侧文档区
1. 右侧「交付物文档」列表
2. 点击「预览」按钮查看文档
3. 可进行以下操作：
   - 📋 复制源码
   - 📝 导出 MD 文件
   - 📄 导出 HTML 文件

### 方式二：从项目目录
1. 点击顶部「📁 项目目录」按钮
2. 选择对应阶段文件夹
3. 点击文件查看

---

## 📌 第七步：项目信息面板

点击右上角「ℹ️ 项目信息」按钮，查看：

- 📋 项目基本信息
- 📅 时间计划
- 📊 项目统计
- ⚙️ 原始数据导出

### 快捷操作按钮
- 💾 **保存进度**：手动保存当前进度
- 🔄 **重置**：重置项目进度（保留文档）
- 📤 **导出**：导出完整项目数据包

---

## 📌 第八步：变更管理（随机触发）

完成任务时，有 20% 概率触发变更事件：

1. 弹出变更请求模态框
2. 查看变更描述和影响评估
3. 点击「批准变更」完成流程
4. 变更会影响进度和成本（模拟）

---

## 📌 第九步：章节解锁

完成当前章节所有任务后，自动解锁下一章节：

```
Ch.1 完成 → 解锁 Ch.2
Ch.2 完成 → 解锁 Ch.3
...
```

---

## 📌 第十步：项目克隆与导出

### 克隆项目
1. 在项目列表中，点击项目卡片的「📋」按钮
2. 系统创建副本（进度重置，名称添加"副本"后缀）
3. 用于重复练习或对比不同方案

### 导出项目
1. 进入项目空间
2. 点击「📤 导出」按钮
3. 下载 JSON 格式的项目数据包
4. 可用于备份或分享

---

## ⚙️ AI 配置（可选）

如需使用 AI 生成文档：

1. 点击顶部「🤖 AI 配置」按钮
2. 填写配置：
   - **Base URL**: `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
   - **Token**: `sk-xxxxx`（从阿里云获取）
   - **模型**: qwen-turbo / qwen-plus / qwen-max
3. 点击「测试连接」验证
4. 保存配置

## 📊 进度说明

### 进度计算
- 总进度 = 已完成任务数 / 总任务数 × 100%
- 每个章节有独立进度
- 实时计算并显示

### 进度保存
- 系统自动保存（localStorage）
- 可手动点击「💾 保存进度」
- 关闭浏览器后重新打开仍保留

## ❓ 常见问题

### Q: 如何重新开始项目？
A: 点击「🔄 重置」按钮，或克隆一个新项目。

### Q: 文档内容可以修改吗？
A: 当前版本为只读预览。如需修改，可导出 MD 文件后用文本编辑器编辑。

### Q: 如何删除项目？
A: 在项目列表中，点击项目卡片的「🗑️」按钮，确认后删除。

### Q: 能否同时创建多个项目？
A: 可以，支持创建无限数量的项目。

### Q: 项目数据存储在哪里？
A: 存储在浏览器的 localStorage 中，每个项目独立存储。
